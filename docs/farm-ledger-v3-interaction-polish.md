# 农事记账小程序 V3 细节打磨路线

## 这版不改首页大结构

当前首页方向可以保留：

- 顶部标题与「记录 / 报表」入口。
- 本月投入主卡。
- 记一笔花费、记一件农事两个主行动卡。
- 今天还没有记录的空状态。
- 底部主导航。

这次重点不是重新排版，而是把控件做得更有质感、更有互动感，并解决地块绑定和输入框遮挡问题。

## 一、按钮升级为「胶囊仓」

现在按钮的问题是：一个个 Chip 分散漂浮，像普通筛选项，缺少统一容器和点击反馈。

建议把分类、地块、农事类型统一设计为「胶囊仓」。

### 胶囊仓结构

```text
你这笔钱花在哪了？

[ 胶囊仓容器 ]
肥料    农药    用工
农机    种苗    其他
```

胶囊仓不是简单一排按钮，而是一个轻容器：

- 外层是浅绿或白色半透明底。
- 内部按钮是圆角胶囊。
- 选中项像「浮起来」一样，有边框、浅阴影和浅绿填充。
- 未选中项弱化，只保留浅灰文字和淡底。

### 推荐样式

外层容器：

- 背景：`rgba(255,255,255,0.72)` 或 `#F7FBF7`
- 圆角：22
- 内边距：8
- 行间距：10
- 可加轻微毛玻璃感，如果小程序性能允许。

胶囊按钮：

- 高度：38 到 42
- 圆角：999
- 横向内边距：16
- 图标尺寸：16 到 18
- 字号：14

选中态：

- 背景：`#E8F7EE`
- 边框：`1px #22B466`
- 文字：`#126C3A`
- 阴影：`0 6 16 rgba(22,139,76,0.12)`
- 图标轻微放大到 1.06 倍。

未选中态：

- 背景：`#FFFFFF`
- 边框：无或 `1px transparent`
- 文字：`#7A857D`
- 阴影：非常轻。

按下态：

- 缩放：`scale(0.96)`
- 背景加深一档。
- 触感上像按下一个软胶囊。

### 分类胶囊不要平均铺满

不要做成整齐网格。大品牌小程序常用「自然换行」：

```text
[肥料] [农药] [用工] [农机]
[种苗] [其他]
```

这样比 2x3 宫格更轻，也更像真实操作控件。

## 二、首页行动卡增加互动感

首页两个大按钮可以保留，但要从「按钮」升级为「行动卡」。

### 记一笔花费

视觉建议：

- 背景使用绿色渐变：左上深一点，右下亮一点。
- 右侧钱袋图标不要孤立，放进一个半透明圆形光斑里。
- 卡片右下角加淡淡的弧形田垄线。
- 点击时整张卡轻微下压。

文案建议：

```text
记一笔花费
买肥、买药、用工、农机
```

### 记一件农事

视觉建议：

- 白色卡片 + 浅绿叶片背景。
- 右侧叶片图标和背景纹理结合，不要像贴纸。
- 选中或按下时边缘出现柔和绿光。

文案建议：

```text
记一件农事
打药、浇水、施肥、巡田
```

### 点击反馈

建议所有大行动卡都有三种状态：

- 默认：轻阴影。
- 按下：卡片下压 2px，阴影变短。
- 跳转：右侧图标轻微向右移动 4px。

## 三、合作社与地块要做成绑定选择

当前「东片一号 / 西片二号 / 全社」像固定选项，不适合合作社真实使用。应该改成「组织 -> 地块」绑定模型。

### 数据关系

```text
合作社 / 主体
  - 地块
    - 小地块 / 棚 / 区域
```

示例：

```text
绿丰合作社
  - 东片一号
  - 东片二号
  - 西片一号

张三家庭农场
  - 一号棚
  - 二号棚
```

### 首页顶部选择

首页标题区建议显示：

```text
今天 · 绿丰合作社
东片一号
```

点击后弹出「地块选择面板」。

### 地块选择面板

不要只做横向 Chip，建议做底部抽屉：

```text
选择记录位置

绿丰合作社                         切换

全部地块
东片一号
东片二号
西片一号

[管理地块]
```

如果用户属于多个合作社：

```text
当前主体
[绿丰合作社 v]

选择地块
[全部地块] [东片一号] [东片二号]
```

### 记账页地块选择

记账页不要把地块裸露成一行文字按钮，改成一个可点击选择卡：

```text
记录位置
绿丰合作社 · 西片二号        >
```

优点：

- 明确钱记到哪个主体。
- 明确钱归属哪块地。
- 后续报表可以按合作社、地块自动汇总。

### 绑定规则

1. 先选合作社 / 主体，再选地块。
2. 地块列表只显示当前主体下的地块。
3. 选择「全社」时，成本记入主体，但不分摊到具体地块。
4. 如果用户只有一个主体，默认不强调主体切换，只显示地块。
5. 如果用户没有地块，显示「先添加地块」引导。

## 四、金额输入框被挡住的问题

截图里金额区域靠上但仍有被内容遮挡或层级混乱的问题。建议从布局和键盘适配两方面处理。

### 页面布局

记花费页结构应固定为三段：

```text
顶部标题区
滚动内容区
底部保存区
```

金额输入区放在滚动内容区靠前位置，但不要被顶部内容压住。

建议顺序：

```text
你这笔钱花在哪了？
分类胶囊仓

金额
[ ¥ 11212 ]

记录位置
[ 合作社 · 地块 > ]

关联农事
[ 可选输入 / 选择 ]
```

### 金额输入视觉

金额输入框建议做成「金额展示卡」，不要像普通输入框。

```text
金额
[ ¥ 11212                 ]
```

样式：

- 高度：72 到 88
- 圆角：18
- 背景：白色
- 金额字号：32 到 38
- 字重：700
- 左侧 `¥` 与数字同一基线。
- 聚焦时卡片边缘出现浅绿光。

### 键盘避挡

底部保存按钮必须考虑键盘。

规则：

1. 键盘弹起时，保存按钮跟随键盘上方，而不是压住内容。
2. 内容区底部留出 `safe-area + 保存按钮高度 + 24` 的滚动空间。
3. 聚焦金额时，自动滚动到金额卡完整可见。
4. 聚焦关联农事或备注时，自动滚动到当前输入框上方留 16 到 24 空隙。

### 不要让底部按钮遮住输入内容

底部保存区建议：

- 白色或半透明渐变底。
- 顶部加 24px 渐隐遮罩。
- 保存按钮高度 54。
- 页面内容底部 padding 至少 110。

## 五、小页面空白处增加背景细节

当前页面空白较多，尤其首页和表单页中下部。不要直接塞更多内容，而是加「弱视觉背景」。

### 首页背景

建议页面背景不是纯色，而是轻渐变：

```text
顶部：#F1F8F2
中部：#F7FAF6
底部：#FFFFFF
```

可以加入：

- 右上角淡绿色圆形光斑。
- 主卡右下角田垄线条。
- 空状态后方淡淡的账本插画。

透明度控制：

- 纹理：5% 到 8%
- 光斑：8% 到 12%
- 插画：12% 到 18%

### 记账页背景

表单页可用「上浅下白」背景：

```text
顶部 0-240px：浅绿雾面渐变
240px 以下：近白背景
```

分类胶囊仓和金额卡悬浮在浅绿背景上，会更有层次。

### 农事页背景

农事页可加入更明显的农业感：

- 顶部右侧淡叶片。
- 备注框右下角淡田垄纹理。
- 底部保存区上方淡白渐隐。

这些细节只做氛围，不要影响读字。

## 六、空状态再升级

当前「今天还没有记录」已经比「暂无记录」好，但还可以更品牌化。

### 首页空状态

```text
今天还没有记录
记下第一笔，月底报表会自动生成

[去记一笔]
```

建议：

- 插画改成「小账本 + 叶片 + 一条田垄线」。
- 按钮不必太大，用绿色小胶囊。
- 空状态整体向上收一点，不要垂直居中过低。

### 记录页空状态

```text
还没有这类记录
换个筛选条件，或者先记一笔花费
```

按钮：

- 记花费
- 记农事

## 七、直接给开发的改动清单

1. 首页大布局保留，不要再重排。
2. 分类、地块、农事类型统一改为「胶囊仓」组件。
3. 胶囊按钮增加选中态、按下态、阴影和缩放反馈。
4. 首页两个主入口改为可按压行动卡，加入渐变、光斑、统一插画。
5. 地块选择改成「合作社 / 主体 -> 地块」绑定结构。
6. 记账页地块显示为「记录位置」选择卡，不再裸露多个固定地块按钮。
7. 金额输入改成大金额卡，并确保聚焦时完整可见。
8. 底部保存按钮适配键盘，不遮挡输入框和关联农事区域。
9. 页面背景加入浅绿渐变、田垄纹理、淡插画，填补空白但不堆内容。
10. 空状态增加插画和轻按钮，避免大片空白像未完成页面。

## 最终效果目标

这版完成后，用户看到的感觉应该是：

- 首页还是熟悉的首页，但更精致。
- 按钮像品牌控件，不像普通标签。
- 合作社和地块归属清楚，不会记错账。
- 输入金额时不被遮挡，保存动作稳定。
- 空白处有呼吸感和农业氛围，不像页面没做完。
