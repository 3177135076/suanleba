# 农事记账小程序 V2 高级感设计路线

## 当前版本的问题

从截图看，当前版本已经比最初三个方案更清晰，但仍然显得普通，主要原因不是功能问题，而是视觉表达太像「表单工具」：

1. 白底 + 绿色按钮过于直白，缺少品牌记忆点。
2. 顶部小程序导航、页内 Tab、底部 Tab 同时出现，层级显得拥挤。
3. 首页空白太多，但空白没有形成高级感，只像内容没填满。
4. 卡片都是普通矩形，缺少主视觉、插画、地块感或经营氛围。
5. 记账弹窗仍然是分类宫格，像后台选择器，不像大品牌小程序的关键动作页。

## V2 方向

建议做成：

## 「农场经营驾驶舱 + 今日事件流」

视觉上靠近大品牌小程序的特点：

- 顶部有品牌化主视觉。
- 首页第一屏有强记忆点。
- 操作入口少但有仪式感。
- 卡片层级更柔和，少边框，多留白和阴影。
- 筛选不做满屏标签，改成轻量胶囊和抽屉。
- 列表不只是记录，而是「今天的农场发生了什么」。

## 视觉关键词

- 清爽
- 有品牌感
- 温和农业感
- 少按钮
- 大卡片
- 轻数据
- 有呼吸感

## 色彩升级

当前绿色太硬，建议调整为「品牌绿 + 稻穗黄 + 雾白背景」。

### 推荐色板

- 页面背景：`#F6F8F3`
- 主品牌绿：`#168B4C`
- 深绿文字：`#145C36`
- 浅绿背景：`#EAF6EF`
- 稻穗黄：`#F4C95D`
- 土壤棕：`#8A623D`
- 主文字：`#1F2A24`
- 次文字：`#7A857D`
- 卡片白：`#FFFFFF`

### 使用方式

- 绿色只用于品牌、主按钮和关键状态，不要到处描边。
- 大面积背景用雾白或浅绿渐变。
- 金额用深绿或黑绿，不要用纯亮绿。
- 农事类型可用低饱和图标色，不要全靠绿色表达。

## 首页重构

### 取消页内三段式 Tab 的强存在感

现在首页、记录、报表的页内 Tab 太像后台。建议：

- 首页顶部不再放大块 Tab。
- 底部导航保留：记账、监测、采集、我的。
- 在「记账」页内部，用顶部右侧入口进入记录和报表。

首页顶部可以这样：

```text
农事记账
今天 · 全社
本月 0 笔 · 农事 0 次              报表
```

### 第一屏做一张品牌主卡

主卡不要只是白色盒子。建议做成带轻渐变和农业纹理的经营卡。

```text
[浅绿渐变经营卡]
本月投入
¥0

今日 ¥0        农事 暂无

一条很淡的田垄纹理 / 地块线条背景
```

视觉重点：

- 卡片高度更大，约 150 到 170。
- 背景使用浅绿渐变。
- 右下角放非常淡的田垄线条、叶片、地块轮廓。
- 金额数字大，但不刺眼。

### 快速入口改成「双行动卡」

现在两个按钮还比较像普通功能入口。建议改成品牌化行动卡：

```text
[主行动卡：记一笔花费]
买肥、买药、用工、农机
右侧：手写账本 / 钱币小插画

[次行动卡：记一件农事]
打药、浇水、施肥、巡田
右侧：叶片 / 田地小插画
```

区别：

- 主行动卡用品牌绿渐变。
- 次行动卡用白卡 + 浅绿背景图形。
- 不使用硬边框，改用轻阴影。

### 最近动态做成「事件流」

空状态不要只写「最近动态」。空状态要有设计：

```text
今天还没有记录
记下第一笔花费，月底报表会自动生成
[去记一笔]
```

配一个淡色账本或田地插画。

有数据后：

```text
今天
09:30  东片 打药
       农药 ¥860

昨天
16:10  西片 施肥
       肥料 ¥3,200
```

不要每条都画完整边框，用时间轴、小圆点和轻分割即可。

## 记一笔花费页

当前弹窗的问题是「分类宫格 + 步骤条」太普通。

建议改成全屏沉浸式记录页，像支付宝、微信支付、Keep 这类小程序的关键动作页。

### 页面结构

```text
记一笔花费                         关闭

你这笔钱花在哪了？

[肥料] [农药] [用工]
[农机] [种苗] [其他]

金额
¥ 0

选择地块     全社
关联农事     可选

[保存]
```

### 关键变化

- 去掉横向步骤条，用户不需要看到「1 类型 / 2 金额 / 3 关联」。
- 标题用问题式文案：你这笔钱花在哪了？
- 分类按钮用大圆角图标胶囊，不用九宫格硬块。
- 金额输入区成为视觉中心。
- 保存按钮固定底部。

## 记一件农事页

农事记录不要像表单，建议像「发布一条农场动态」。

```text
记一件农事

今天做了什么？

[打药] [浇水] [施肥]
[除草] [采收] [巡田]

在哪块地？
[东片] [西片] [全社]

补充说明
例如：雨后补打、浇水 2 小时

顺便记成本
[添加金额]

[保存]
```

这会比步骤式流程更自然。

## 记录页重构

当前记录页筛选 Chip 太多，视觉上像筛选后台。

### 推荐方式

顶部只保留三个轻量筛选：

```text
2026年7月    全社    筛选
```

点「筛选」后再展开：

- 类型
- 地块
- 成本 / 农事

列表用日期分组，不要一排排空 Chip。

### 空状态

```text
暂无记录
记一笔花费或农事后，这里会按时间自动整理
[记一笔] [记农事]
```

## 报表页重构

当前报表页像几个空白卡片。建议变成「经营结论页」。

### 首屏卡片

```text
7月经营概览
¥0
本月投入

0 笔成本       0 次农事
```

### 成本构成

空数据时不要显示空卡标题，显示引导：

```text
还没有成本构成
记录肥料、农药、用工后，会自动生成占比
```

有数据时用品牌化进度条：

```text
肥料        ¥3,200      42%
[==========]
农药        ¥860        11%
[===]
```

### 地块对比

用地块卡片，不用表格：

```text
东片
¥4,200
本月 5 笔

西片
¥2,800
本月 3 笔
```

## 大品牌感的关键细节

### 1. 少边框，多层次

当前很多卡片靠绿色边框区分，显得廉价。建议：

- 主卡片用背景色和渐变。
- 次级卡片用阴影和间距。
- 只有选中态才用描边。

### 2. 图标统一风格

现在图标像不同来源拼在一起。建议统一：

- 线性图标 + 轻填充。
- 图标颜色不超过 4 种。
- 记账、监测、采集、我的用同一套视觉。

### 3. 空状态要有情绪价值

不要只写「暂无记录」。改成：

- 今天还没记账
- 记下第一笔，月底报表自动算
- 配淡色插画

### 4. 让农业属性更明显

不用加复杂插画，可以加很轻的元素：

- 田垄线条
- 叶片纹理
- 地块轮廓
- 稻穗小图形

这些元素透明度控制在 5% 到 12%，不要抢内容。

### 5. 底部导航更现代

当前底部导航图标有点散。建议：

- 图标统一线性。
- 当前 Tab 使用浅绿圆形背景。
- 文案字号更小、更克制。
- 底部顶部线改成非常淡，不要用强绿色线。

## V2 与当前截图的直接改动清单

1. 首页移除页内大 Tab，减少顶部拥挤感。
2. 首页主卡改成浅绿渐变经营卡，增加田垄纹理。
3. 「记一笔花费」从普通绿色按钮升级为带插画的主行动卡。
4. 「记一件农事」从描边卡升级为白色品牌行动卡。
5. 最近动态空状态增加插画和引导按钮。
6. 记账弹窗改为沉浸式全屏记录页，去掉步骤条。
7. 记录页减少横向 Chip，改为「月份 / 地块 / 筛选」。
8. 报表页空状态改为经营结论引导，不显示空白卡片。
9. 底部导航图标统一，弱化顶部绿线。
10. 全局减少硬描边，增加渐变、阴影、插画和留白。

## 最终建议

不要追求更复杂，而是追求更有品牌感：

- 首页像经营看板。
- 记录像农场日记。
- 记账像一次轻松的支付动作。
- 报表像经营结论。

这版会比当前截图更接近微信、支付宝、美团、盒马这类大品牌小程序的视觉节奏：少控件、强主卡、轻筛选、好空状态、有品牌图形。
